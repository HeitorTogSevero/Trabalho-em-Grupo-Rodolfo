Teoria dos Conjuntos

A teoria dos conjuntos é um ramo da matemática que estuda a coleção de objetos, chamados de elementos. Os elementos (que podem ser qualquer coisa: números, pessoas, frutas) são indicados por letras minúsculas. Já os conjuntos são representados por letras maiúsculas e, normalmente, seus elementos aparecem entre chaves { }, separados por vírgula.

Exemplo: A = {a, e, i, o, u}

Representação dos Conjuntos

Um conjunto pode ser representado de algumas maneiras, dependendo da situação em que é usado:

  • Diagrama de Venn: Os conjuntos são representados graficamente por figuras planas fechadas (como círculos ou elipses) que "guardam" seus elementos. São excelentes para visualizar elementos repetidos ou conjuntos sem nada em comum (disjuntos).
  • Tabular: Utiliza as chaves { } para listar os elementos explicitamente.
    Exemplos: A = {1, 3, 9, 12, 17} ou B = {João, Luíza, Fernando, Lívia}
  • Propriedade: O conjunto é definido por uma regra ou característica comum de seus elementos.
    Exemplo 1: A = {x | x é uma vogal} (Lê-se: "O conjunto A é formado pelos elementos x, tal que x é uma vogal").
    Exemplo 2: B = {x ∈ N; | x é múltiplo de 3} (Lê-se: "O conjunto B é formado pelos elementos x, tal que x pertença aos números naturais e seja um múltiplo de 3").

Relações Básicas

(Elemento → Conjunto)

Indica se um componente específico faz parte ou não de um determinado conjunto. É uma relação exclusiva entre elemento e conjunto.

  • ∈ (Pertence): Se D = {w, x, y, z}, então w ∈ D (w pertence ao conjunto D).
  • ∉ (Não pertence): Usando o mesmo conjunto anterior, j ∉ D (j não pertence ao conjunto D).

Relação de Inclusão (Conjunto → Conjunto)

Aponta a relação de subconjuntos existentes entre dois ou mais conjuntos:

  • ⊂ (Está contido): Todos os elementos do primeiro conjunto estão no segundo.
  • ⊆ (Não está contido): Pelo menos um elemento do primeiro conjunto não faz parte do segundo.
  • ⊃ (Contém): O conjunto maior possui o conjunto menor dentro dele.
Exemplo de Inclusão:
Se A = {a, e, i, o, u} e B = {a, e, i, o, u, m, n, o}, temos que A ⊂ B (A está contido em B) e B ⊃ A (B contém A).

Conjunto Vazio

O conjunto vazio é aquele que não possui nenhum elemento. Ele é representado por duas chaves sem nada dentro { } ou pelo símbolo ∅. Importante: O conjunto vazio está contido (⊂) em todos os conjuntos.

União, Intersecção e Diferença entre Conjuntos

As operações fundamentais ajudam a solucionar uma grande variedade de problemas matemáticos:

União de Conjuntos (A ∪ B)

Corresponde à junção dos elementos de dois ou mais conjuntos, sem repetir os elementos comuns. No diagrama de Venn, ela é representada pelo preenchimento de toda a imagem.

Se A = {1, 2, 3, 4} e B = {3, 4, 5, 6} → A ∪ B = {1, 2, 3, 4, 5, 6}

Intersecção de Conjuntos (A ∩ B)

Corresponde apenas aos elementos que se repetem (são comuns) em ambos os conjuntos. No diagrama, é representada pintando unicamente a área de sobreposição (o meio).

Se C = {a, b, c, d, e} e D = {d, e, f, g, h} → C ∩ D = {d, e}

Diferença de Conjuntos (A - B)

É o conjunto formado pelos elementos que estão no primeiro conjunto, mas não aparecem no segundo. No diagrama, pinta-se apenas a parte exclusiva do conjunto da esquerda.

Se A = {a, b, c, d, e} e B = {b, c, d, f, g} → A - B = {a, e}

Atenção: A ordem importa, portanto A - B ≠ B - A. No exemplo acima, B - A seria igual a {f, g}.

Igualdade de Conjuntos

Dois conjuntos são considerados idênticos (A = B) quando possuem exatamente os mesmos elementos, mesmo que eles estejam escritos em uma ordem diferente.

Exemplo: {1, 2, 3, 4, 5} = {3, 5, 4, 1, 2}
