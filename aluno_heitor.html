Função do 2º Grau (Função Quadrática)

A Função do 2º Grau, também conhecida como Função Quadrática, é uma função matemática muito utilizada para representar trajetórias, lucros, áreas e diversos fenômenos do cotidiano.

f(x) = ax² + bx + c

Onde:

  • a, b e c são números reais.
  • a ≠ 0.
  • O maior expoente da variável x é 2.

O que significam a, b e c?

  • a: determina a abertura e a direção da parábola.
  • b: influencia a posição do vértice.
  • c: indica onde o gráfico corta o eixo Y.

Exemplo de Função

f(x) = x² - 4x + 3

Essa é uma função do 2º grau porque o maior expoente de x é 2.

Como é o gráfico?

O gráfico de uma função quadrática sempre forma uma curva chamada parábola.

Concavidade da Parábola

  • Se a > 0, a parábola fica voltada para cima.
  • Se a < 0, a parábola fica voltada para baixo.

Quanto maior o valor absoluto de "a", mais fechada será a parábola.

Vértice da Parábola

O vértice é o ponto mais importante da parábola, pois representa seu ponto máximo ou mínimo.

  • Parábola para cima → vértice é o ponto mínimo.
  • Parábola para baixo → vértice é o ponto máximo.

Raízes da Função

As raízes são os pontos onde a parábola toca ou corta o eixo X.

x = (-b ± √Δ) / 2a

Essa fórmula é conhecida como Fórmula de Bhaskara.

Discriminante (Δ)

Antes de aplicar Bhaskara, calculamos o discriminante:

Δ = b² - 4ac
  • Δ > 0 → duas raízes reais.
  • Δ = 0 → uma raiz real.
  • Δ < 0 → não possui raízes reais.

Exemplo Resolvido

f(x) = x² - 5x + 6

Identificando os coeficientes:

  • a = 1
  • b = -5
  • c = 6

Calculando Δ:

Δ = (-5)² - 4·1·6
Δ = 25 - 24
Δ = 1

Como Δ é positivo, a função possui duas raízes reais.

Analogia

Imagine uma bola sendo chutada para o alto. Ela sobe, atinge uma altura máxima e depois desce. Esse movimento tem formato parecido com uma parábola e pode ser representado por uma função do 2º grau.

Aplicações no Dia a Dia

  • Trajetória de bolas e projéteis.
  • Construção de pontes e arcos.
  • Cálculo de lucros e prejuízos.
  • Problemas de Física.
  • Jogos, animações e simulações.

Curiosidade

Muitos arcos arquitetônicos possuem formato semelhante ao de uma parábola. Além disso, a trajetória de uma bola de futebol também pode ser aproximada por uma função quadrática.

Resumo

  • Forma geral: ax² + bx + c.
  • O coeficiente a deve ser diferente de zero.
  • O gráfico é uma parábola.
  • Possui vértice, raízes e concavidade.
  • As raízes podem ser encontradas pela Fórmula de Bhaskara.
  • É muito utilizada na Matemática, Física e Engenharia.
